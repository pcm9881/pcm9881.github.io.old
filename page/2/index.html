Mozilla Observatory

웹사이트 보안 스캔 후 피드백 해주는 사이트 observatory

보안 검증을 원하는 사이트 스캔

main

  • Don’t include my site in the public results
    검색하는 사이트를 공개 목록에 공유하기 싫으면 선택하세요.
  • Force a rescan instead of returning cached results
    24시간내에 캐시된 결과가 아닌 강제로 새로 검색 하려면 선택하세요.
  • Don’t scan with third-party scanners
    Mozilla Observatory 외에 타사에서 제공하는 스캔기능을 사용하지 않으려면 선택하세요.
  1. enter domain name here 란에 자사 사이트 또는 개인 사이트 입력
  2. Scan Me 클릭

PostgREST

PostgREST는 PostgreSQL 데이터베이스를 RESTful API로 직접 변환하는 독립형 웹 서버. postgres

  • 설명

GET http://postgrest.example.com/ member ?id=eq.1
Verb (HTTP METHOD) RESOURCE (URI) Representation 1 Representation 2

select

예를 들어서 컬럼을 구별에서 뽑고 싶으면 select=컬럼1,컬럼2를 쿼리로 주면 된다.
이때 따로 명시를 안하면 모두 * (asterisk)으로 표현되서 가져온다.

  • 예시
    member 테이블에서 id가 1인 id 와 name 컬럼을 가져와라

count

select=count.*

  • 예시
    member 테이블에서 id가 1인 데이터 수를 가져와라

in

?id=in.(1,2,3)

  • 예시
    member 테이블에서 id가 1,2,3 인 데이터를 가져와라
