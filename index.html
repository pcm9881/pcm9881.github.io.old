virtualbox-install-ubuntu

Posted on 3월 4, 2019
  1. VirtualBox 설치
  2. Download Ubuntu ISO
  3. BIOS 에서 Virtualization option 이 enabled 되어있는지 확인
  4. VirtualBox 로 Ubuntu 설치 (설치방법은 검색하면 나옴)
  5. 필수 Ubuntu package 설치
    sudo apt-get install build-essential
  6. Ubuntu 띄워진 상태에서 메뉴의 Insert Guest Additions 클릭하여 설치
  7. 재부팅시 화면이 Full Screen 으로 돌아가는지 확인
  8. VirtualBox 의 Setting 에서 Disk Size, CPU, RAM (memory size) 등 조정
  9. VirtualBox 의 Setting 에서 Shared Clipboard, Drag & Drop 을 bidirectional 으로 설정

Posted on 3월 4, 2019

PostgREST

PostgREST는 PostgreSQL 데이터베이스를 RESTful API로 직접 변환하는 독립형 웹 서버. postgres

Query

  • 예시

  • 설명
    앞에 HTTP METHOD , Postgrest URI, 테이블명 , 조건으로 설명.

GET http://postgrest.example.com/ member ?id=eq.1
Verb (HTTP METHOD) RESOURCE (URI) Representation 1 Representation 2
GET, POST, PUT, DELETE PostgREST 운영중인 URI 테이블명 조건

where

select

예를 들어서 컬럼을 구별에서 뽑고 싶으면 select=컬럼1,컬럼2를 쿼리로 주면 된다.
이때 따로 명시를 안하면 모두 * (asterisk)으로 표현되서 가져온다.

  • 예시
    member 테이블에서 id가 1인 id 와 name 컬럼을 가져와라

count

select=count.*

  • 예시
    member 테이블에서 id가 1인 데이터 수를 가져와라

in

?id=in.(1,2,3)

  • 예시
    member 테이블에서 id가 1,2,3 인 데이터를 가져와라

Reference

  1. PostgREST 커스텀 쿼리 : http://postgrest.org/en/v5.2/api.html#custom-queries
